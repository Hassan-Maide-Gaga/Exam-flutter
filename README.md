# BadWallet Mobile

Application mobile de gestion de portefeuille électronique BadWallet.

## Technologies
- Flutter 3.x
- Provider (State Management)
- HTTP/Dio (API Client)
- flutter_secure_storage (Stockage sécurisé)

## Fonctionnalités
- ✅ Authentification simplifiée par numéro de téléphone
- ✅ Affichage du solde avec masquage
- ✅ Transfert d'argent entre utilisateurs
- ✅ Paiement de factures (ISM, WOYAFAL, SENELEC, etc.)
- ✅ Historique des transactions avec filtres
- ✅ Interface moderne et responsive
